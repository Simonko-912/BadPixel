Yes, these apps are vibecoded, but i personaly use them to make this font so i added thease apps.
First run the html one and load the bdf if you want to edit the font, then (since every single char from menu is added) add the characters you want using the custom option and enter the unicode number.
Simply by drawing it saves the char real time, and if you want to contribute, export the .h and .bdf file and spritesheet image and run the conventer python app to convert the .h to bdf
Then make a pr and add the .h .bdf, .ttf and the spritesheet image and say what unicode chars you added.
